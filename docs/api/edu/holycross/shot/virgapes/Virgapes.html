case class Virgapes(domain: CtsUrn) extends MidOrthography with Product with Serializable

An orthographic system for neumes encoded in the Virgapes encoding, registered to a specific text or set of texts identified by CtsUrn.

A set of one or more texts (depending on the hierarchical level of domain) identified by a CtsUrn.
